# Dark Mode Toggle

A simple and responsive dark mode toggle implemented with HTML, CSS, and vanilla JavaScript.

## Live Demo
[https://dark-mode-project-three.vercel.app/](https://dark-mode-project-three.vercel.app/)

## Requirements Checklist
- [x] Toggle persists across reload (localStorage)
- [x] Respects `prefers-color-scheme` on first load
- [x] Smooth color transition
- [x] All components themed via CSS variables

## Features
- **Persistent:** Remembers user preference using `localStorage`.
- **System Preference:** Detects user OS theme on first visit.
- **No FOUC:** Included theme script in head to prevent flash of wrong theme.

## Technologies Used
- HTML5
- CSS3 (Variables & Transitions)
- JavaScript

## How to Run Locally
1. Clone this repository: `git clone https://github.com/AMRIsoja/dark-mode-project`
2. Open `index.html` in your browser.
